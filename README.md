# AI Agents Course

Learning repository for **AI Agents with OpenAI, CrewAI, LangGraph, AutoGen, and MCP**.

This project captures my course notes, labs, and implementations focused on building AI agents that go beyond simple chatbots. The work here explores autonomous behavior, tool usage, multi-agent collaboration, workflow orchestration, and external system integration.

## Overview

In this course, I learned how to design and build AI agents that can perform tasks autonomously using large language models (LLMs). The main focus was on creating intelligent, tool-integrated, and collaborative agent systems for real-world use cases.

## Key Concepts Learned

### AI Agents Fundamentals

- Reasoning, decision-making, memory, and tool usage
- The **ReAct** pattern: **Reason -> Act -> Observe**
- Core building blocks of autonomous agent behavior

### OpenAI Agents SDK

- Built single agents with tool calling
- Generated structured outputs in JSON
- Automated research and workflow tasks

### CrewAI

- Designed multi-agent systems with role-based collaboration
- Built teams of agents such as:
  - Researcher
  - Analyst
  - Writer
- Improved task execution through delegation

### LangGraph

- Created stateful workflows with loops and retries
- Managed agent state and memory persistence
- Built systems for more complex agent logic

### AutoGen

- Implemented agent-to-agent communication
- Built collaborative agent teams
- Enabled coordinated problem-solving flows

### MCP

- Connected agents to external tools and APIs
- Built modular and scalable integrations
- Used a standardized protocol for tool access
